# 🤖 ROS2 Robotics Lab 🚀

A collection of robotics projects built using **ROS2 Humble**, Python, ESP32, Arduino, sensors, and real-time visualization systems.

This repository documents my robotics learning journey through practical ROS2-based projects involving:

* Sensor integration
* Autonomous robot logic
* Real-time communication
* Visualization systems
* Hardware & simulation workflows

---

# 📦 Projects Included

---

## 🔹 1. ROS2 Distance Monitoring System

A real-time ROS2-based distance monitoring system using an HC-SR04 ultrasonic sensor and Arduino/ESP32.

### ✅ Features

* Real-time distance measurement
* ROS2 Publisher & Subscriber communication
* Obstacle detection logic
* Serial communication with microcontroller
* Distance filtering for invalid readings

### 🛠️ Technologies Used

* ROS2 Humble
* Python
* HC-SR04 Ultrasonic Sensor
* Arduino / ESP32
* Ubuntu 22.04

---

## 🔹 2. Autonomous Obstacle Avoiding Robot

A ROS2-based autonomous robot decision system capable of making movement decisions based on obstacle distance.

### ✅ Features

* Autonomous navigation logic
* Multi-node ROS2 architecture
* Movement decision system
* STOP / MOVE_FORWARD / SLOW_FORWARD logic
* TURN_LEFT / TURN_RIGHT / MOVE_BACKWARD actions
* Simulation-first workflow

### 🛠️ Technologies Used

* ROS2 Humble
* Python
* ESP32
* Motor Driver (L298N)
* Ultrasonic Sensor

### 🤖 Robot Behaviors

* 🚗 Move Forward
* 🐢 Slow Forward
* 🛑 Stop
* ↩️ Turn Left
* ↪️ Turn Right
* ⬅️ Move Backward

---

## 🔹 3. ROS2 Radar Visualization System

A real-time radar visualization system built using ROS2 and Pygame.

The system simulates a radar sweep and visualizes obstacle distance and angle data dynamically.

### ✅ Features

* Real-time radar visualization
* Radar sweep animation
* Obstacle plotting
* ROS2 topic communication
* Angle & distance tracking
* Pygame-based GUI

### 🛠️ Technologies Used

* ROS2 Humble
* Python
* Pygame
* ESP32 / Arduino
* HC-SR04 Sensor
* Servo Motor
